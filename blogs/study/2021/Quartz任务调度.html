Gen 2021-05-24 quartz

# Scheduler Interface

调度程序维护JobDetails 和Tirgger的注册表。注册后，调度程序负责在相关触发器触发时(当它们的计划时间到达时)执行作业。

调度程序实例由调度程序工厂生成。已经创建/初始化的调度程序可以通过生产它的同一工厂找到并使用。创建调度程序后，它处于 stand-by mode （待机模式)，必须先调用start()方法，然后才能启动任何作业。

任务将由客户程序通过定义一个实现Job接口的类来创建。然后创建JobDetail 对象(也由客户端创建)，以定义Job的单个实例。JobDetail实例可以通过scheduleJob(JobDetail, Trigger)或addJob(JobDetail, boolean) 方法向调度程序注册。

可以根据给定的计划定义触发器来触发单个Job实例。SimpleTrigger（简单触发器）对于一次性触发最有用，或者在一个精确的时刻触发，在它们之间有给定延迟的N次重复。CronTrigger（Cron触发器）允许基于一天中的时间、一周中的日期、一个月中的日期和一年中的月份进行调度。

存储在调度器中的Job也可以通过使用 triggerJob(String jobName, String jobGroup) 功能“手动”触发。

Quartz提供了对Job实例监听的Listener接口。JobListener接口提供作业执行的通知。TriggerListener 接口提供触发器触发的通知。SchedulerListener 接口提供调度器事件和错误的通知。侦听器可以通过ListenerManager接口与本地调度程序相关联。

这是Quartz调度器的接口信息，省略了不重要或者说是没有使用过的接口。

工件清单 - 调度接口

  • start() ：启动触发触发器的调度程序线程。
  • shutdown()：停止调度程序触发触发器，并清除与调度程序相关联的所有资源。
  • standby()：调度器处于待机模式，需要调用start方法开启。
  • scheduleJob()：将给定的作业详细信息添加到调度程序中，并将给定的触发器与之相关联。
  • unscheduleJob()：从调度程序中删除指示的触发器。TODO:如果任务已经运行，删除是否还有效吗？
  • rescheduleJob()：移除(删除)具有给定关键字的触发器，并存储新的给定触发器，它必须与同一作业相关联(新触发器必须具有指定的作业名称和组)。但是，新触发器不需要与旧触发器具有相同的名称。
  • deleteJob()：从调度程序删除已存在的Job实例，以及和它关联的触发器。
  • triggerJob()：触发已存在调度器中的JobDetails(现在执行)。
  • pauseXXXX()：暂停触发Job实例。
  • resumeXXX()：恢复触发Job实例的执行。
  • interrupt(JobKey jobKey)：在此调度程序实例中，请求中断已识别作业的所有当前执行实例，该实例必须是可中断作业接口的实现者。

# 设计一个任务调度的状态机

# 任务的约束条件

任务的操作是进行信息采集，在任务开始前需要执行一些准备操作后进行数据采集操作，在结束采集时，需要关闭采集数据。因此，一个采集任务涉及到两个Job实例，且这些Job实例仅需要触发一次即可，不需要周期地触发。

在操作的过程中不需要进行暂停和恢复的操作，只需要提供立即运行，调度，停止操作即可。

立即运行任务，需要提供任务的结束时间。

调度任务，需要提供任务的开始和结束时间。

任务的状态有等待，调度，运行，完成，中断，失败。

# 任务接口设计

任务实例（Job实现类）需要了解这些信息

  • 任务标识，jobkey
  • 任务组，jobGroup
  • 任务执行类全限定类名
  • 时间，立即运行的任务，是结束时间，调度的任务是开始时间
  • 克隆表达式，仅调度任务是需要
  • 任务类型，开始采集准备任务，结束采集收尾任务
  • 任务状态，等待，调度，运行，完成，中断，失败
  • 可执行的操作，立即运行，调度，停止

public class JobInfo implements Serializable {

    private static final String DEFAULT_GROUP = "DEVICE";
    private static final Logger log = LoggerFactory.getLogger(JobInfo.class);

    private String taskIdentity;
    private String taskGroup;
    private String cron;
    private String time;
    private String taskClazzName;
    private TaskStateEnum state;

    @Deprecated
    public final static JobInfo build(Task task, JobType type) {
        JobInfo jobInfo = new JobInfo();
        jobInfo.taskIdentity = type.name() + "_" + task.getTaskName();
        jobInfo.taskGroup = DEFAULT_GROUP;
        try {
            if (type == JobType.START) {
                jobInfo.cron = buildCron(task.getBeginTime());
            } else {
                jobInfo.cron = buildCron(task.getEndTime());
            }
        } catch (ParseException e) {
            log.info("error : "+ e.getMessage());
            throw new ServiceException(e.getMessage());
        }
        return jobInfo;
    }

    public static  String buildCron(String time) throws ParseException {
        SimpleDateFormat dateFormat = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
        SimpleDateFormat cronFormat = new SimpleDateFormat("s m H d M ? yyyy");

        Date date = dateFormat.parse(time);
        return cronFormat.format(date);
    }


    public static JobInfo.JobBuilder builder() {
        return new JobInfo.JobBuilder();
    }


    public static class JobBuilder {
        private  String taskIdentity;
        private  String taskGroup;
        private  String cron;
        private  String time;
        private  String taskClazzName;
        private  TaskStateEnum state;

        public JobBuilder setJobType(JobType type, String taskKey) {
            if (StringUtils.isEmpty(type) || StringUtils.isEmpty(taskKey)) {
                throw new ServiceException("参数不能为空");
            }
            taskIdentity = type.name() + "_" + taskKey;
            return this;
        }

        public JobBuilder setTime(String execTime) {
            if (StringUtils.isEmpty(execTime)) {
                throw new ServiceException("参数不能为空");
            }
            try {
                this.time = execTime;
                cron = buildCron(execTime);
            } catch (ParseException e) {
                log.error("error : "+ e.getMessage());
                throw new ServiceException(e.getMessage());
            }
            return this;
        }

        public JobBuilder setClassName(String className) {
            if (StringUtils.isEmpty(className)) {
                throw new ServiceException("参数不能为空");
            }
            taskClazzName = className;
            return this;
        }

        public JobBuilder setGroup(String taskGroup) {
            if (StringUtils.isEmpty(taskGroup)) {
                throw new ServiceException("自定义参数值不能为空");
            }
            this.taskGroup = taskGroup;
            return this;
        }

        public JobBuilder setState(TaskStateEnum state) {
            this.state = state;
            return this;
        }

        public JobInfo build() {
            JobInfo info = new JobInfo();
            info.taskIdentity = taskIdentity;
            info.cron = cron;
            info.time = time;
            info.taskGroup = DEFAULT_GROUP;
            info.taskClazzName = taskClazzName;
            info.state = state;
            return info;
        }
    }

    public String getTaskGroup() {
        return taskGroup;
    }

    public String getTaskIdentity() {
        return taskIdentity;
    }

    public String getCron() {
        return cron;
    }

    public String getTime() {
        return time;
    }

    public String getTaskClazzName() {
        return taskClazzName;
    }

    public TaskStateEnum getState() {
        return state;
    }

    public void setState(TaskStateEnum state) {
        this.state = state;
    }

    @Override
    public String toString() {
        return "JobInfo{" +
                "taskIdentity='" + taskIdentity + '\'' +
                ", taskGroup='" + taskGroup + '\'' +
                ", cron='" + cron + '\'' +
                ", taskClazzName='" + taskClazzName + '\'' +
                '}';
    }
}

# Quartz调度器逻辑封装

代码清单 - QuartzManager 
public class QuartzManager {

    private static Logger log = LoggerFactory.getLogger(QuartzManager.class);

    private final Scheduler scheduler;
    private static final String JOB_DATA_KEY = "jobInfo";

    public QuartzManager(Scheduler scheduler) {
        this.scheduler = scheduler;
        try {
            if (!scheduler.isShutdown()) {
                scheduler.start();
            }
        } catch (SchedulerException e) {
            throw new ServiceException("调度器启动失败");
        }
    }

    private void checkJobInterface(Class<?> taskClazz, String className) {
        if (!Job.class.isAssignableFrom(taskClazz)) {
            log.error(className + " 没有实现org.quartz.Job接口");
            throw new ServiceException(className + "没有实现org.quartz.Job接口");
        }
    }

    private Class<?> getClassOf(String className) {
        try {
            return Class.forName(className);
        } catch (ClassNotFoundException e) {
            log.error("获取实现类的字节码对象失败");
            throw new ServiceException("获取实现类的字节码对象失败");
        }
    }

    public void schedule(JobInfo jobInfo) {
        log.info("开始调度任务：{}", jobInfo.getTaskIdentity());
        String className = jobInfo.getTaskClazzName();
        Class taskClazz = getClassOf(className);
        checkUserInterface(taskClazz, className);

        String jobIdentity = jobInfo.getTaskIdentity();
        JobDataMap newJobDataMap = new JobDataMap();
        newJobDataMap.put(JOB_DATA_KEY, jobInfo);

        // 任务执行信息
        JobDetail jobDetail = JobBuilder.newJob(taskClazz)
                .withIdentity(jobIdentity, jobInfo.getTaskGroup())
                .setJobData(newJobDataMap)
                .build();

        // 任务的时间触发器
        TriggerBuilder<Trigger> triggerBuilder = TriggerBuilder.newTrigger()
                .withIdentity(jobIdentity, jobInfo.getTaskGroup());

        triggerBuilder.withSchedule(CronScheduleBuilder.cronSchedule(jobInfo.getCron()));
        CronTrigger trigger = (CronTrigger) triggerBuilder.build();

        try {
            // 加入调度池，等待调度,根据触发器设置的规则执行任务
            scheduler.scheduleJob(jobDetail, trigger);
        } catch (SchedulerException e) {
            log.error("尝试触发 {} 任务失败", jobInfo.getTaskIdentity());
            throw new ServiceException(e.getMessage());
        }
    }


    /**
     * 直接调度一次任务 (任务不在调度池)
     */
    public void startAtNow(JobInfo jobInfo) {
        String className = jobInfo.getTaskClazzName();
        Class taskClazz = getClassOf(className);
        checkUserInterface(taskClazz, className);

        String jobIdentity = jobInfo.getTaskIdentity();
        JobDataMap newJobDataMap = new JobDataMap();
        newJobDataMap.put(JOB_DATA_KEY, jobInfo);

        // 任务执行信息
        JobDetail jobDetail = JobBuilder.newJob(taskClazz)
                .withIdentity(jobIdentity, jobInfo.getTaskGroup())
                .setJobData(newJobDataMap)
                .build();

        // 任务的时间触发器
        TriggerBuilder<Trigger> triggerBuilder = TriggerBuilder.newTrigger()
                .withIdentity(jobIdentity, jobInfo.getTaskGroup())
                .startNow()
                .endAt(parse(jobInfo.getTime()));

        SimpleTrigger trigger = triggerBuilder
                .withSchedule(SimpleScheduleBuilder.simpleSchedule())
                .build();

        try {
            scheduler.scheduleJob(jobDetail, trigger);
        } catch (SchedulerException e) {
            log.error("任务调度异常：{}", e.getMessage());
        }
    }

    private Date parse(String time) {
        if (!Optional.ofNullable(time).isPresent()) {
            throw new ServiceException("时间为空字符串");
        }
        SimpleDateFormat format = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
        try {
            return format.parse(time);
        } catch (ParseException e) {
            throw new ServiceException("时间解析异常:" + time);
        }
    }



    /**
     * 停止调度一个任务
     */
    public void shutdown(JobInfo jobInfo) {
        log.info("停止调度任务");
        String jobIdentity = jobInfo.getTaskIdentity();
        TriggerKey triggerKey = TriggerKey.triggerKey(jobIdentity, jobInfo.getTaskGroup());
        try {
            // 停止触发器
            scheduler.pauseTrigger(triggerKey);
            // 移除触发器
            scheduler.unscheduleJob(triggerKey);
            // 删除任务
            scheduler.deleteJob(JobKey.jobKey(jobIdentity, jobInfo.getTaskGroup()));
        } catch (Exception e) {
            log.error("任务 {} 不存在", jobIdentity);
            throw new SystemException(ResultCode.INTERNAL_SERVER_ERROR);
        }
    }

    /**
     * 前置条件：任务处于调度或运行的状态，任务接口需要实现Job和InterruptableJob接口
     *
     * 停止任务
     *
     * 后置条件：任务执行终止或调度终止
     *
     * + 执行分支
     *   - 可以通过再次调度任务，使任务运行起来
     *
     */
    public void interrupt(JobInfo jobInfo) {
        log.info("执行中断任务");
        String jobIdentity = jobInfo.getTaskIdentity();
        TriggerKey triggerKey = TriggerKey.triggerKey(
            jobIdentity, 
            jobInfo.getTaskGroup()
        );
        JobKey jobKey = JobKey.jobKey(jobIdentity, jobInfo.getTaskGroup());
        try {
            // 停止触发器
            scheduler.pauseTrigger(triggerKey);
            // 移除触发器
            scheduler.unscheduleJob(triggerKey);
            // 中断任务
            scheduler.interrupt(jobKey);
            // 删除任务
            scheduler.deleteJob(jobKey);
        } catch (Exception e) {
            log.error("任务{}停止失败：{}", jobIdentity, e.getMessage());
            throw new SystemException(ResultCode.INTERNAL_SERVER_ERROR);
        }
    }

}

public class JobExecutor implements Job , InterruptableJob {

    private Logger log = LoggerFactory.getLogger(getClass());

    private int count = 0;

    @Override
    public void execute(JobExecutionContext jobExecutionContext) 
        throws JobExecutionException {
        try {
            for (; count < 1000; count ++) {
                System.out.println("执行任务：......" + this.toString());
                TimeUnit.SECONDS.sleep(1);
            }
        } catch (InterruptedException e) {
            log.error("中断任务发生异常：{}", e.getMessage);
        }
    }

    @Override
    public void interrupt() throws UnableToInterruptJobException {
        count = 1000;
    }
}

public class WaitStateTask implements TaskStateStrategy {

    private TaskStateMachine machine;

    public WaitStateTask(TaskStateMachine machine) {
        this.machine = machine;
    }

    @Override
    public void schedule(JobInfo job, QuartzManager manager) {
        manager.schedule(job);

        machine.setCurrentState(machine.getRunning());
        job.setState(TaskStateEnum.RUNNING);
    }

    @Override
    public void shutdown(JobInfo job, QuartzManager manager) {

    }
}

public class TaskStateMachine {

    private final Logger log = LoggerFactory.getLogger(getClass());

    private TaskStateStrategy wait;
    private TaskStateStrategy schedule;
    private TaskStateStrategy running;

    private QuartzManager manager;
    private TaskStateStrategy currentState;

    public TaskStateMachine(QuartzManager manager) {
        wait = new WaitStateTask(this);
        schedule = new ScheduleStateTask(this);
        running = new RunningStateTask(this);
        this.manager = manager;
        this.currentState = wait;
    }

    public boolean schedule(JobInfo jobInfo) {
        log.info("当前状态：" + jobInfo.getState());
        currentState.schedule(jobInfo, manager);
        return false;
    }

    public boolean forthRun(JobInfo jobInfo) {
        log.info("当前状态：" + jobInfo.getState());
        currentState.forthRunning(jobInfo, manager);
        return false;
    }

    public boolean shutdown(JobInfo jobInfo) {
        log.info("当前状态：" + jobInfo.getState());
        currentState.shutdown(jobInfo, manager);
        return false;
    }

    public void setCurrentState(TaskStateStrategy currentState) {
        this.currentState = currentState;
    }

    public TaskStateStrategy getRunning() {
        return running;
    }

    public TaskStateStrategy getSchedule() {
        return schedule;
    }

    public TaskStateStrategy getWait() {
        return wait;
    }
}

public class ParentCaseTester {

    protected final Logger logger = LoggerFactory.getLogger(getClass());

    /**
     * for example:
     *      nowAfter(1, ChronoUnit.MINUTES);
     *      返回现在时间延后1分钟的时间
     *
     * @param amount
     * @param temporalUnit
     * @return
     */
    protected String nowAfter(long amount, TemporalUnit temporalUnit) {
        LocalDateTime nowTime = LocalDateTime.now();
        LocalDateTime beginTime = nowTime.plus(amount, temporalUnit);
        DateTimeFormatter formatter = DateTimeFormatter.
            ofPattern("yyyy-MM-dd HH:mm:ss", Locale.CHINA);
        return beginTime.format(formatter);
    }
}
