打印1到最大的n位数

Genlg 2020-10-17 剑指offer

题目：输入数字n，按顺序打印出从1最大的n位十进制数。比如输入3，则打印出1、2、3一直到最大的3位数即999。

# 关键词

  • 大数问题
  • 模拟加法
  • 递归
  • 交互习惯

# 自己实现的思路

首先对输入值n的范围做合法校验，其次初始化char数组用于存储数字，初始化为‘0’。接着开始判断是否达到最大的数值，如果没有，输出char数组中的信息。

public char[] printMaxValue(int slots) {
    // at first, check slot range
    if (slot <= 0) {
        return null;s
    }s
    char[] num = new char[slots];
    initialized(num);
    while (increment(num)) {
        print(num);
    }
    return num;
}

/**
 * implement addition for char array
 *
 * @param num
 * @return
 */
private boolean isIncrement(char[] num) {

    for (int len = num.length, i = len - 1; i >= 0; i--) {
        // check max value in every slot of array : time  O(N)
        boolean isIncrement = false;
        for (int j = 0; j < len; j++) {
            isIncrement = num[j] == '9' ? (isIncrement | false) : (isIncrement | true);
        }
        if (!isIncrement) {
            return false;
        }
        // mock add method for char array
        num[i] += 1;
        if (num[i] <= '9') {
            break;
        } else {
            num[i] = '0';
            continue;
        }
    }
    return true;
}

private void initialized(char[] num) {
    for (int i = 0, size = num.length; i < size; i++) {
        num[i] = '0';
    }
}

// 按剑指offer的输出要求
private void print(char[] number) {
    boolean isZero = true;
    for (int i = 0, len = number.length; i < len; i++) {
        // 短路效应
        if (isZero && number[i] != '0') {
            isZero = false;
        }
        if (!isZero) {
            System.out.print(number[i]);
        }
    }
    System.out.println();
}

参考剑指offer的解答，做出一些优化，如下

private boolean increment(char[] number) {

    for (int i = number.length - 1; i >= 0; i--) {
        // carry one for value of current slot
        number[i] += 1;

        if (number[i] <= '9') {
            break;
        }
        if (i == 0) {
            // the high is over flow then return that can not increment for number
            return false;
        } else {
            // the pre high need add one and current slot change zero
            number[i] = '0';
        }
    }
    return true;
}

# 递归实现

public void printMaxValue(int n) {
    if (n <= 0) {
        return;
    }
    char[] number = new char[n];
    recursively(number, 0);
}

/**
 * number           万千百十个
 *        | * * * * * * * * * |
 * slot     0 1 2 3 4 5 6 7 8
 *
 * @param number
 * @param slot
 */
private void recursively(char[] number, int slot) {
    // bounder then print all char information
    if (slot >= number.length) {
        print(number);
        return;
    }
    /*
     * do a step : current slot set value then require next slot
     * range in  [0 - 9]
     */
    for (int i = 0; i <= 9; i ++) {
        number[slot] = (char) ('0' + i);
        recursively(number, slot + 1);
    }
}

# 本题扩展

在前面的代码中，我们都是用一个char型字符表示十进制数字的一位。8个bit的char型字符最多能表示256个字符，而十进制数字只有0～9的10个数字。因此用char型字符串来表示十进制的数字并没有充分利用内存，有一些浪费。有没有更高效的方式来表示大数？
