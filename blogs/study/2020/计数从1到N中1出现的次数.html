# 计数从1到N中1出现的次数

# 题目描述

题目：输入一个整数n，求从1到n这n个整数的十进制表示中1出现的次数。例如输入12，从1到12这些整数中包含1 的数字有1，10，11和12，1一共出现了5次。

# 题目考察哪些素质

  • 寻找规律的能力
  • 分析总结能力

# 题目分析

通过举例子寻找数字中1出现的次数F与范围n之间的关系，也就是F(n) = f(n) (n >=0)

首先使用数组来分解数字中的每个位，用n标识索引。接下来使用145来作为案例

范围 1出现的次数 公式总结
1 ~ 9 1 10^0 * (0 + 1)
1 ~ 99 20 10^1 * (1 + 1)
1 ~ 999 300 10^2 * (2 + 1)
1 ~ 10^n - 1 -- 10^(n-1) * n

此时，总结出在1~10^n - 1之间，一共出现**10^(n-1) * n **个1

根据目前掌握的规律可以得出1~99之间有20个1

接下来，发现100 ~ 145之间的数字规律

![](/otsma/images/计算100 - n.png)

  • 百位上如果等于1的情况下，在这个区间内它出现了145 - 100 + 1 = 46 个 1，百位上大于1的情况与这种计算方式不同，因此，另外进行讨论。
  • 接下来我们需要看1 ~ 45里面有多少个1，这其实是一种递推的关系，还需要使用一个数组存储45中有多少个1，以及5中有多少个1，暂时用c数组来标识当前有多少个1。45里面有1的个数位 c[1]，这其实是自底向上的计算，我在分析的时候是自顶向下的。

在上述发现中，可以看出，数字数组a中的每个位都有两种情况需要讨论：

  • 当前索引上的数字等于1
  • 当前索引上的数字大于1

当前讨论的是百位上的数字等于1的情况

现在总结一下，需要计算的值来自三部分

当a[2] = 1时

① 1 ~ 99 10^1 * 2 = 20

② 100 ~ 145 145 - 100 + 1 = 46

③ 1 ~ 45 c[1]

表达式为a[2] = ① + ② + ③

推广至n，得： $$ c[n] = 10 ^ (n-1) * n + \Sigma_{m=0}^n (a[m] * 10 ^ m) - 10 ^ n + 1 + c[n-1] ,,,,,,,,,,, (c[n] = 1 ,,, n >= 1) （1） $$

现在开始分析45中有多少个1

  • 墨蓝色标识的是1 ~ 9 中出现的1的次数 为 1
  • 10 ~ 40中十位出现的1的次数为10，黑色标识的部分
  • 10 ~ 40中个位出现的1的次数为3，紫蓝色标识的部分
  • 1 ~ 5中只出现过1个1，绿色标识的部分，即c[0]的值

描述10 ~ 40之间的规律，还需要一个辅助案例来寻找通用的规律

以数字345为例，寻找100 ~ 300之间出现1的频次

综上所述，得出总结如下：

当a[1] > 1时

① 1 ~ 9 10^0 * 1 = 1

② 10 ~ 40 (a[1] - 1) * 10^0 + 10^1

③ 1 ~ 5 c[0]

表达式为a[1] = ① + ② + ③

通过总结，可以进一步进行分析，发现 1 ~ 9 和 10 ~ 40可以整合成 *a[1] * 10^0 1

推广至n，并整理公式： $$ c[n] = a[n]*10^(n-1)*n + 10^n + c[0],,,,,,,,,,, (c[n] > 1 ,,, n >= 1) （2） $$

整理（1）（2）式，得： $$ f(a,n) = \begin{cases} 10 ^ (n-1) * n + \Sigma_{m=0}^n (a[m] * 10 ^ m) - 10 ^ n + 1 + f(a,n-1) ,,,,,,,,,,, (a[n] = 1 ,,, n >= 1)\ \ a[n]*10^(n-1)*n + 10^n + f(a,n-1),,,,,,,,,,, (a[n] > 1 ,,, n >= 1)\ \end{cases} $$ 通过公式可以看出，使用代码实现有两种方式：

  • 递归
  • 递推

# 代码实现

# 1. 递推实现（按照自己分析思路实现）

public int countTimesOfOne(int number) {

    int preCount = number % 10 == 0 ? 0 : 1;
    int n = 1;
    int currentNumber;
    int pow = 1;
    int subNumber = number % 10;
    number = number / 10;

    while (number != 0) {
        currentNumber = number % 10;

        if (currentNumber == 1) {
            preCount = pow * n + subNumber + 1 + preCount;
        } else if (currentNumber > 1){
            int x = pow * n;
            preCount = x + 10 * pow + (currentNumber - 1) * x + preCount;
        }
        pow *= 10;
        n ++;
        subNumber = subNumber + currentNumber * pow;
        number = number / 10;
    }
    return preCount;
}
