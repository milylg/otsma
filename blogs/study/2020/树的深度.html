计算二叉树的深度

Gen 2020-10-24 剑指offer

题目一：输入一棵二叉树的根结点，求该树的深度。从根结点到叶结点依次经过的结点（含根、叶结点）形成树的一条路径，最长路径的长度为树的深度。

# 题目分析

二叉树的父节点有左右节点，利用递归的性质获取左右子树的高度，首先获取左右子节点的高度，在判断左右子树的最大高度，然后加一，标识当前节点的高度。

# 代码实现

方式一

    public static int depthOf(Node root) {
        // 边界条件-递结束，开始归
        if (root == null) {
            return 0;
        }
        int leftChildDepth = depthOf(root.left);
        int rightChildDepth = depthOf(root.right);
        return leftChildDepth > rightChildDepth ? leftChildDepth + 1 : rightChildDepth + 1;
    }

方式二

private static int depthOf(Node root, int level) {
    if (root == null) {
        return level - 1;
    }
    int leftChildDepth = depthOf(root.left, level + 1);
    int rightChildDepth = depthOf(root.right, level + 1);
    return leftChildDepth > rightChildDepth ? leftChildDepth : rightChildDepth;
}

这两种方式的不同之处在于计算树的高度是自顶向下还是自底向上。

方式一在左右子节点都遍历完成后，也就是触发了边界条件之后，遍历到树的叶子节点，执行高度比较的操作时计数节点的高度，最大的高度值不断的归到上层调用函数中，返回给左右节点高度的记录器中（leftChildDepth and rightChildDepth）。

方式二在遍历根节点的时候，就计数高度为1，传入根节点的高度，然后每层函数调用时都在当前节点的高度下递增一层，传递到下一个节点，直到节点为null，触底反弹，开始执行高度比较操作，选取当前节点最高的高度。

上述其实都是后序遍历的框架，只是计算操作的时机不同，前者时自底向上递增，后者时自顶向下递增高度。

# 扩展题

题目二：输入一棵二叉树的根结点，判断该树是不是平衡二叉树。如果某二叉树中任意结点的左右子树的深度相差不超过1，那么它就是一棵平衡二叉树。

这题其实用上面的代码做一个递归判断即可，只是每次递归都使用depthOf方法，效率有点低。

# 如何优化

  • 寻找重复计算的地方
  • 针对重复计算的部分进行优化

上面的代码中遍历到一个节点，都要计算两次树的深度计算，该节点下的一些子节点被重复访问了。此处是代码需要优化的地方。

在自底而上的归的过程中记录当前节点的最大高度，并用于上一层调用函数的判断数据。

# 优化代码

private class Depth {
    int depth;
    public Depth(int depth) {
        this.depth = depth;
    }
}

public boolean  isBalanceTree(Node root) {
    final Depth depth = new Depth(0);
    return isBalanceTree(root, depth);
}

private boolean isBalanceTree(Node root, Depth depth) {
    if (root == null) {
        return true;
    }
    Depth left = new Depth(0), right = new Depth(0);
    if (isBalanceTree(root.left, left) && isBalanceTree(root.right, right)) {
        int diff = left.depth - right.depth;
        if (diff <= 1 && diff >= -1) {
            depth.depth = left.depth > right.depth ? left.depth + 1 : right.depth + 1;
            return true;
        }
    }
    return false;
}
